# 📊 Pharmaceutical Sales Analytics Dashboard

An interactive Streamlit dashboard for exploring pharmaceutical sales trends across drug categories (ATC codes).

## 🔗 Data Source
Dataset from Kaggle: [Pharmaceutical Sales Data](https://www.kaggle.com/datasets/milanzdravkovic/pharma-sales-data)

##  Files
| File | Description |
|------|-------------|
| `app.py` | Streamlit dashboard app |
| `salesdaily.csv` | Daily sales data |
| `saleshourly.csv` | Hourly sales data |
| `salesweekly.csv` | Weekly sales data |
| `salesmonthly.csv` | Monthly sales data |
| `Pharma_sales_data_kaggle.ipynb` | Exploratory data analysis notebook |

##  How to Run

1. Install dependencies:
```bash
   pip install -r requirements.txt
```

2. Run the app:
```bash
   streamlit run app.py
```

##  Requirements
- pandas
- matplotlib
- streamlit
